## vdo 微度
vdo是什么?
> 这是一个Java相关技术的实践代码仓库，  
> 基于Maven构建的多模块工程
> 每个模块的README.md文件会对模块进行具体说明     

具体会写些什么?
> 1.某种技术的QuickStart;  
> 2.一个具体的Demo实例;  
> 3.总结和见解;  
> X...  

模块的命名规则  
> xxx技术的QuickStart -> xxx-start  
> xxx技术的Demo -> xxx-demo  
> xxx技术的总结和见解 -> xxx-learn
> ...
